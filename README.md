# 🇦🇷 Countryball Revelión

Este juego lo inventó y diseñó **Max, un nene de 8 años**. La idea, las reglas y los personajes son 100% suyos — el código se escribió para hacer realidad lo que imaginó.

🎮 **Jugalo acá:** https://sidneyrees.github.io/juego-max/

## De qué se trata

Manejás a `countryballmax`, el countryball de Argentina (🇦🇷 el mejor countryball de la historia, según su creador). Tiene que sobrevivir comiendo frutas **nomnom** 🍎 y **pomnom** 🍐, y tomando **bebidas** 🥤 para no quedarse sin hambre ni sed. Si las barras llegan a cero, se muere.

Para hacerla más difícil, hay un **fantasma** 👻 estilo Pac-Man dando vueltas: si te toca, quedás congelado 5 segundos sin poder moverte.

## Cómo jugarlo

- **Compu:** con las flechas del teclado (⬆️⬇️⬅️➡️).
- **Celular:** con los botones en pantalla, o instalando la app (es una PWA — desde Chrome, menú ⋮ → "Instalar aplicación") para jugar sin necesitar internet.

## Hecho con

HTML, CSS y JavaScript puro (sin librerías), dibujado todo con el `<canvas>`.
